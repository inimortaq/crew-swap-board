One-time setup — about 10 minutes
1
Create a new Google Sheet. Go to sheets.google.com and create a blank sheet. Name it "Crew Swap Board".
2
Add two sheets inside it. Rename the first tab to Swaps and create a second tab called Pickups.

In Swaps row 1, add these headers:
Timestamp Name Type Flight Date Duration Pax Wants WantsDates WantsAfter WantsBefore AlsoDrop ALDates Notes Contact RowID
In Pickups row 1, add these headers:
Timestamp Name Flight Duration Dates Notes Contact RowID
3
Open Apps Script via Extensions → Apps Script. Delete any existing code, paste in the script below and click Save (Cmd+S).
4
Deploy the script. Click Deploy → New deployment. Set type to Web app. Set "Who has access" to Anyone. Click Deploy, authorise when prompted, then copy the web app URL.
5
Paste the URL here and click Save.
6
You're done! Go to the Swap Board tab and click Refresh to confirm it's connected. Share the URL of this page with your crew.
